關於我

您好！我是熱愛程式設計的軟體工程師，自 2023 年起積極投入軟體開發領域。

為了系統性學習，我參加了職訓局的「Java 程式與網頁設計班」，學習了 Java、HTML、CSS、JavaScript、jQuery、Bootstrap、MySQL 等基礎技術。

結訓後，我進入職場擔任 Java 為主的網頁開發工作，負責公司網站的開發與維護，實務中運用了多項技術，如 Java、Ajax、Linux、MongoDB、PHP 等。

之後，我進一步參加「Swift 程式設計與網頁設計班」，開始學習 Swift，也接觸了 Flutter、Kotlin、AI 工具應用、更深入的 Linux 系統管理與 Apache 部署等內容，持續拓展技能樹。
