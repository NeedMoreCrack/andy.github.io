關於我

您好！  我是熱愛程式設計的工程師，自2023年起積極投入這個領域。
為了系統性的學習程式開發，我參加了職訓局的「Java程式與網頁設計班」，並在在這段訓練期間學習了Java程式語言，以及前後端開發的基礎知識，如HTML、CSS、JavaScript、jQuery、Bootstrap、MySQL等技術。
結訓後進入職場，擔任 Java 為主的網頁開發與維護 工作，負責公司網站的開發與維護，並在實務中應用了多項技術，包括 Java、HTML、CSS、JavaScript、jQuery、Bootstrap、MySQL、Ajax、Linux、MongoDB、PHP 等。
為了進一步拓展技能，我後續參加了 職訓局「Swift 程式設計與網頁設計班」，開始學習 Swift，並持續充實自己的技術背景。此外，在這段期間，我也接觸到了更多的 AI 工具應用、更深入的 Linux 系統管理與 Apache 部署，以及 Flutter 和 Kotlin 等語言，拓展了更多的開發技能。
